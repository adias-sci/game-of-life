# Colorful Life Adventure with Interactive Panels

This version incorporates interactive pieces:
- Animated spinner panel
- Event popups for payday, taxes, action cards, family, houses, career, retirement
- House shopping with card-style buttons
- Career and house panels
- Floating money burst animation
- Colorful winding board
- Online room codes using Supabase
- Single-player and pass-and-play

## Setup

1. In Supabase, run `life-supabase-setup.sql`.
2. Upload files to GitHub.
3. Deploy on Vercel.
4. Share the Vercel link. Host creates a code, players join with the code.

## Google Sites

Google Sites → Insert → Embed → By URL → paste your Vercel URL.
