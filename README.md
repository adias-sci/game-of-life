# Life Deluxe V3

A more polished, colorful Life-style browser game with:

- Curved colorful board
- Animated spinner wheel
- Guided action center
- Player dashboard cards
- Clickable player profiles
- Career deck
- Action card flip panel
- House showroom
- Life Tile gallery
- Retirement island
- Floating money animations
- Single-player with bots
- Pass-and-play
- Supabase online room codes

## Online setup

1. Supabase → SQL Editor
2. Paste `life-v3-supabase-setup.sql`
3. Run it once
4. Upload the files to GitHub
5. Deploy on Vercel
6. Share the Vercel URL. Host creates a code; players join with it.

## Google Sites

Insert → Embed → By URL → paste your Vercel URL.

This is an original Life-style game, not an official licensed product.
